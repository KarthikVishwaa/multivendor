# MultiVendor Ecommerce Platform 🛒

A full-stack MultiVendor Ecommerce platform built with modern web technologies.

This application allows multiple vendors to manage and sell products while customers can browse, purchase, and manage orders through a responsive and modern user interface.

---

# 🚀 Features

## Customer Features
- User authentication
- Browse products
- Product categories
- Add to cart
- Secure checkout
- Order management
- Responsive shopping experience

## Vendor Features
- Vendor dashboard
- Add/Edit/Delete products
- Product image uploads
- Inventory management
- Order handling

## Admin Features
- Manage vendors
- Manage products
- Monitor orders
- Platform management

---

# 🛠️ Tech Stack

## Frontend
- React.js
- Tailwind CSS
- TypeScript

## Backend
- Node.js
- Express.js
- MongoDB
- Mongoose

## Payment & Uploads
- Stripe Payment Gateway
- Cloud/Image Upload Support
